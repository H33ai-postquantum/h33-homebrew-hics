# Homebrew Tap for HICS

```bash
brew tap h33/tap https://gitlab.com/h33-ai/homebrew-hics
brew install hics
```

Then:

```bash
hics scan .
```

Free. Local. No account. No network calls.

## What is HICS?

H33 Independent Code Scoring. Five security dimensions. STARK-proven. Dilithium-signed. The algorithm is the authority.
